# TCP/IP (Protocolo de Controle de Transmissao)

 - E um protocolo de comunicaçao entre computadores em rede.
 - Possui 4 camadas:
    - Fisica (PLACA DE REDE)
    - Rede (IP)
    - Transporte (
        - ## TCP
            - Voltado a conexao
            - Handshake (verifica a conexao entre o destino e origem, 'conexao privada')
            - Integridade, ordem dos dados
            - Aplicativo de mensagens de texto
        
        - ## UDP
            - Rapido
            - Nao confiavel
            - Carro do ovo ('conexao publica')
            - Livestream
        )
    - Aplicaçao (FTP, HTTP)
        - ## Portas
            ##### Por onde dados sairao e chegarao


## Roteadores
 - Dispositivos 'burros' (envia a informaçao para todos e quem realmente a pediu pega a mesma)

## Switch
 - Dispositivos 'inteligentes' (envia a informaçao apenas para quem pediu a mesma)
