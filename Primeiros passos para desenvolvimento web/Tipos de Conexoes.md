# Wi-Fi

Possui alguns padroes, sao eles:
 - IEEE 802.11: 2,4 GHz, 2Mbps
 - IEEE 802.11a: 5 GHz, 54Mbps
 - IEEE 802.11b: 2,4 GHz, 11Mbps (menos interferencia)
 - IEEE 802.11g: 2,4 GHz, 54Mbps
 - IEEE 802.11n: 2,4 GHz/ 5GHZ, 150-600 Mbps

 ## Segurança
  - WEP;
  - WPA;
  - WPA2
    - Mais seguro
    - Requer maior precessamento

# Bluetooth
