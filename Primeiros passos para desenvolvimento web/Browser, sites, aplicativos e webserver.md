# Browser

Existem muitas formas de se acessar a Internet, mas para ver um site, eh preciso um programa que interprete as linguagens de programaçao e as transforme em algo compreensivel para um humano.

# Sites

Sao paginas web que atendem diversos propositos. Podem ser desenvolvidos em diversas linguagens de programaçao.

# Aplicativo

Eh um software executado no navegador. Ou seja, um aplicativo de celular nada mais eh do que uma especie de navegador.
Atualmente quase nao existe diferença entre site e aplicativo (sites estao em declinio).

# E-commerce (Comercio Eletronico)

Site de compra e venda com sistema de pagamento, este pode ser exterior ao site.

# Web-server

Existem dois tipos de web-server, sao eles:
 - ## Estatico
    Eh um servidor fisico onde sao armazenados arquivos, softwares e/ou banco de dados.
  - ## Dinamico: 
    Se refere aos softwares que estao presentes no servidor fisico (file server, application server, database)

Os dados e banco de dados de um site ou aplicativo precisam estar em um servidor.

# Web-Service

Interface disponivel para fazer requisiçoes e consultas em banco de dados inacessiveis (Correios, Governo)

[Link para Correios Web Service](https://cws.correios.com.br/ajuda) - plataforma de serviços web, que permite acesso, por integração, aos dados que podem auxiliar o cliente dos Correios em funcionalidades que visa agregar mais valor ao seu produto ou serviço e com dados mais assertivos, a exemplo dos dados de endereçamento.
