# Stacks
     Ambiente tecnologico, com todas suas ferramentas e capacidades disponiveis para gerar soluçoes digitais

- Pilhas de Tecnologia;
- Conjunto de softwares necessarios e suficientes p executar um app/programa;
- Linguagens de Programaçao;
- Ambientes (produçao, homologaçao, desenvolvimento...) e ferramentas de interaçao com o app/sw;
- Capacidade e limitaçoes de performance;
- Pontos fortes e fracos do app/sw

## Front-end

Stack de desenvolvimento que trabalha com a 'parte de frente' do site, software, app, web service. Lida com a interface, UI, UX utilizando logica de programaçao, HTML, CSS, PHP, Bootstrap, jQuery e/ou outros frameworks.

## Back-end

Stack de desenvolvimento que trabalha com a 'parte de tras' da aplicaçao. Lida com servidores, banco de dados dentre outras coisas, atuando como 'meio de campo' entre interface e banco de dados, regra de negocios, validaçoes fazendo uso do MySql, Oracle, protocolos de comunicaçao, PHP, Java, Node.js, dentre outros.

## Fullstack

Profissional que sabe trabalhar em todas as camadas das tecnologias de desenvolvimento/execuçao de um app/sw
